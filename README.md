# TUI todo application

## Good to know for those who try it out
Add task reminder takes following formats:
   "%Y-%m-%d %H:%M",
   "%d/%m/%Y %H:%M",
   "%d-%m-%Y %H:%M",
   "%Y-%m-%d",
   "%d/%m/%Y",
   "%d.%m.%Y",

And: 
  "1d 2h",
  "1h",
  "tomorrow",
  "next week",
  "today",
